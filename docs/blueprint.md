## Core Features:

- Geolocation Check: Automatically verify user location against predefined geographical boundary.
- Name and ID Input: Accepts the user's name and ID for check-in on the initial screen.
- Automatic Check-In: Automatically check-in the user when their location is verified within the specified geofence.
- Activity Log: Store all check-in activities including timestamp, location and name.

## Style Guidelines:

- Primary color: Vibrant orange (#FF4500) to reflect the Shopee brand identity.
- Background color: Light orange (#FFE4B2) for a warm and inviting feel.
- Accent color: Darker orange (#FF8C00) to highlight important elements and calls to action.
- Body and headline font: 'Roboto' sans-serif font for clear readability and a modern aesthetic.
- Use icons that align with the Shopee brand, simple and easily recognizable.
- Clean and straightforward layout to ensure ease of use.
- Smooth transitions and loading animations to create a pleasant user experience.
